# Biosphere Project 🌎
##  Overview
This project aims to desing and study a small-scale biosphere system using an aquariu, as a controlled enviroment. 
##  Objective
To undertand how biological, physical, and eviromental variables interact in a semi-closed ecosystem. 
## System Description
- Type: Semi-closed aquatic system
- Componets:
  - Water
  - Living organism (fish, plants, microorganism)
  - Light as energy input
## Variables to Monitor
- Temperature
- Light exposure
- Water clarity
- Biological activity
## Research Questions
- How does light intensity affect algea growth?
- How stable is the system over time?
- What factors influence equilibrium?
## Tools & Technology
- Arduino
- Sensors
- Manual observation logs
